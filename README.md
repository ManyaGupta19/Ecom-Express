# Ecom Express – Logistics Dashboard (Power BI)

## Overview
This project analyzes logistics performance using Power BI, focusing on shipment trends and delivery efficiency across regions.

## Objective
To identify delivery delays, understand regional performance, and improve operational decision-making.

## Key Insights
- Certain regions consistently show higher delivery delays  
- Delivery efficiency drops during peak order periods  
- A small number of regions contribute to most delays  
- Uneven shipment distribution highlights resource imbalance  

## Tools Used
- Power BI  
- Data Visualization  

## Outcome
The dashboard helps identify operational gaps and supports targeted improvements in logistics planning and efficiency.
